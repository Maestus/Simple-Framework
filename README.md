Build project :

    make

--

How to use :

  For 2048 :
    ./launch 2048 size_x size_y [--graphic]

  For Taquin :
    ./launch Taquin size_x size_y [--graphic]

  For Sokoban :
    ./launch Sokoban size_x size_y [--graphic]

--
